Installation
============

Target group: **Administrators**

.. important::

   This extension is made for TYPO3 CMS 9.x and 10.x LTS.

The extension needs to be installed as any other extension of TYPO3 CMS:

Latest version from git
-----------------------
You can get the latest version from git by using the git command:

.. code-block:: bash

   git clone git@github.com:carsten-walther/html2pdf.git

.. important::

   The master branch supports latest TYPO3 CMS only.

Preparation: Include static TypoScript
--------------------------------------

This is not needed. Configuration is made in the Extension Configuration module.

Reports
-------

To check whether all server-side settings have been made, you can check the settings in the 'Reports' module.
